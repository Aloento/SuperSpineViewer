# SuperSpineViewer
一个拿来加载与导出Spine动画的工具  
至少需要Java15才能运行  
当前版本：0.1.7 测试版  

## 关于这个工具
它本质上是我学习Java和OpenJFX的练习作品  
所以我虽然尽可能保证它能正常使用  
但是不能保证它好用，也不能保证拥有大多数功能  

### 为什么用Java
其实已经有一个很好的Viewer了，叫SpineViewerWPF  
它是用C#写的，能实现的功能应该也会比我的多吧  
但是要安装.NET Framework，这玩意不能跨平台  
所以我纠结以后决定拿Java来写一个全平台的  
更何况，Spine不就是拿Java写的嘛  

### 关于引用
本程序内含大量来自网络的代码和他人的库  
具体有哪些我也列不出来  
反正就是有很多，毕竟是个人学习作品  

## 功能
### 目前实现的
加载Spine导出的Skel和Json  
查看与播放动画  
查看与切换皮肤  
变换大小  
调整位置  
调整速度  
循环播放  
播放暂停  
导出透明MOV  
导出透明PNG
